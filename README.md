# AI-Powered Collections Strategy

AI-powered delinquency risk analysis and collections strategy simulation completed as part of the Tata Forage Data Analytics Virtual Experience Program.

---

## Project Overview

This project focused on:
- Exploratory Data Analysis (EDA)
- Delinquency risk assessment
- Predictive modeling concepts
- Responsible AI strategy design
- Business-focused collections recommendations

---

## Key Risk Indicators

- High Credit Utilization
- Multiple Missed Payments
- High Debt-to-Income Ratio

---

## Predictive Modeling Approach

- Logistic Regression
- Risk classification framework
- Performance evaluation using Accuracy, Recall, F1-Score, and AUC-ROC

---

## Responsible AI Guardrails

- Fairness monitoring
- Explainable predictions
- Human-in-the-loop oversight
- Ethical financial decision-making

---

## Business Recommendations

- Targeted outreach for high-risk customers
- Automated reminder systems
- AI-assisted collections prioritization

---

## Technologies & Concepts

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- EDA
- Logistic Regression
- Business Analytics
